## Installation

```console
composer require shel/neos-resource-import-preprocessor
```

## Configuration options

Copy the following configuration into your `Settings.yaml` to adjust the plugin's behavior:

```yaml
Shel:
  Neos:
    ResourceImportPreprocessor:
      adjustFilename:
        enabled: true
      autoScale:
        enabled: true
```
